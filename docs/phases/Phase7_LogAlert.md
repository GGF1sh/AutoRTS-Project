## 背景（ユーザー要望）

> ログは色分けして（ユーザーも色分けできるように）。特定キャラだけ／敵だけ／味方だけ抜粋できる表示切替。特定の行動で音、HP一定値以下でアラーム、あるいは一時停止 ―― ガンビット系だとそういうのが大事。

## 実装内容

### 1. ログの色分け
- `RichTextLabel` の BBCode を有効化し、種類ごとに色付け。
- 色は `Main.gd` 冒頭の `LOG_COLORS` 辞書を書き換えるだけで変更可能（ユーザーが自分で色分け可）。
  - 攻撃=グレー / 回復=緑 / 後退=黄 / 撃破=赤 / 警告=オレンジ / システム=水色

### 2. ログ表示フィルタ（画面右上パネル）
- 陣営: 味方 / 敵 / システム のチェックボックス
- 種類: 攻撃 / 回復 / 後退 / 撃破 のチェックボックス
- 対象キャラ: OptionButton で「全員 or 特定キャラ」を選んで抜粋
- ログは履歴 `MAX_LOG_HISTORY=300` 件を保持し、フィルタ後の最新 `MAX_LOG_LINES=12` 行を表示。

### 3. HP低下アラート
- `low_hp_threshold`（既定 30%）を**下回った瞬間だけ**発火（毎フレーム鳴らない）。
- 画面上部に「⚠ <名前> HP低下！」を表示し、Tween でフェードアウト。
- 回復でしきい値を上回ると警告フラグがリセットされ、再度下回れば再通知。

### 4. 音（外部ファイル不要）
- `AudioStreamWAV` に正弦波ビープをコードで合成（880Hz, 0.18秒, 減衰あり）。素材ファイル不要で動く。
- 右上「アラーム音」チェックでON/OFF。

### 5. 自動一時停止
- 右上「HP低下で一時停止」チェックON時、アラート発火で `is_paused=true`。
- ガンビット調整のため、危険な瞬間に自動で止められる。

## 技術メモ / 注意

- チェックボックス類は `focus_mode = FOCUS_NONE`。スペースキーを奪わせず、一時停止トグルを邪魔しないため。
- フィルタUIは `CanvasLayer` 上にあり一時停止の影響を受けない（停止中も操作・確認できる＝狙いどおり）。
- CheckBox のハンドラは `Callable.bind()` でキーを渡す方式。1行ラムダで複数文を書く不安定さを回避。

## 既知の制約 / 次への申し送り

- **MP はまだデータモデルに無い**。MPアラームは MP を使うユニットを作る時に同じ仕組みで拡張する。
- 色や音量、しきい値の**ゲーム内設定UI**は未実装（今はコード定数）。設定画面は将来。
- ログの「行動」種類は現状 後退/回復 のみ詳細ログ。攻撃対象選択などの細かい判断ログは今後追加可能。
