# v0.9.0 — biim式レイアウト＋エフェクト/効果音＋ユニット拡張＋キャッシュ最適化

- **日付**: 2026-06-15 / **担当**: Claude / **状態**: 実装完了（実機確認待ち）

## 背景

ユーザー要望（UIが戦場に被って見づらい／ログをもっと見たい／地味／種類を増やしたい）と、Gemini・Grok のコードレビュー指摘を反映。

## 実装内容

### 1. biim式レイアウト（UIを戦場に被せない）
- ウィンドウを 1280×720 に拡大。
- `battle_rect`（戦場の矩形）を定義し、枠（`_draw` で塗り＋ボーダー）で囲う。
- 右＝設定パネル、下＝戦闘ログ、と**戦場の外側**に配置。ユニットは `battle_rect` 内にだけ配置・後退も枠内にクランプ。

### 2. ログのバックログ化
- 表示行数の制限を撤廃し、全履歴（最大400件）を `RichTextLabel` に表示。`scroll_following` で最新に追従しつつ、**一時停止中に上へスクロールして遡れる**。
- ログ欄も画面下いっぱいに拡大（高さ210px）。

### 3. 攻撃/回復のエフェクト＆効果音
- 攻撃：攻撃者→対象に色付きライン＋着弾ドット（`Main._draw` の `_effects` を時間減衰）。
- 回復：対象に緑のリングが広がる。
- 効果音：攻撃＝低い短音、回復＝中音。`AudioStreamWAV` をコード合成（素材不要）。`効果音` チェック（`sfx_on`）でON/OFF、プリセット対応。

### 4. ユニット種類の拡張
- 味方5体（タロウ/ゴウ(重装)/レン/ミナ/アキラ）、敵5体（レイダー/スカウト/スナイパー/敵衛生兵/ブルート）。
- `data/units.json` 編集で増減可。新ロール `Scout`（高速・弱点狙い）を `gambits.json` に追加。**敵衛生兵**で敵側の回復も観察できる。

### 5. パフォーマンス（レビュー反映）
- Gemini/Grok 指摘の「`get_nodes_in_group` 毎フレーム多用」を改善。`Main` が生存ユニットを**毎フレーム1回キャッシュ**し、各 Unit は `battle.get_alive_units()` を参照（ユニット増に備える）。
